= Bookies

== DESCRIPTION:

Bookies is a Rails plugin for managing cookies for Facebook Connect.

== USAGE:

# Initialize Bookies
bookies = Bookies.new(cookies, @app_id, @app_secret, @host)

# Get User Data
bookies[:uid]
bookies[:session_key]
bookies[:access_token]
bookies[:expires]

# Clear Facebook cookies
bookies.clear!
